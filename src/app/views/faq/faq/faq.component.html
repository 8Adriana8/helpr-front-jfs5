Para criar um novo chamado, na página HOME, clique no menu localizado na barra superior esquerda, representado pelo ícone menu. Deslize até o botão CHAMADOS e clique no ícone: perm_phone_msg. Você será direcionado(a) para a página de chamados. No canto superior direito, clique no add para abrir o campo de novo chamado. Após preencher as informações requiridas, clique em SALVAR.

Para editar um chamado, na página HOME, clique no menu localizado na barra superior esquerda, representado pelo ícone menu. Deslize até o botão CHAMADOS e clique no ícone: perm_phone_msg. Você será direcionado(a) para a página de chamados. Sobre o chamado que deseja editar, clique no botão edit para abrir o campo de edição. Após realizar as alterações desejadas , clique em SALVAR.

Para filtrar um chamado, na página HOME, clique no menu localizado na barra superior esquerda, representado pelo ícone menu. Deslize até o botão CHAMADOS e clique no ícone: perm_phone_msg. Você será direcionado(a) para a página de chamados. Acima da tabela de chamados, há uma linha, denomidada FILTER. Clique na linha e digite as palavras-chave de sua busca. Automaticamente, aparecerá o chamado correspondente.
